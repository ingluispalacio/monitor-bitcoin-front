¿Cómo funcionan los Feature Toggles?

Los Feature Toggles te permiten activar o desactivar funcionalidades específicas de la plataforma en tiempo real, sin necesidad de redeplegar la aplicación. Esto es útil para:

  • Mantenimiento: Desactiva módulos durante actualizaciones
  • Pruebas A/B: Activa características gradualmente
  • Control de acceso: Limita funcionalidades por período
  • Respuesta rápida: Desactiva módulos problemáticos inmediatamente

💰 Módulo de Compras

Control de órdenes de clientes

  • Permite/Blquea: Creación de nuevas órdenes por clientes
  • Útil para: Mantenimiento nocturno, pausas por volatilidad
  • Comportamiento: Las órdenes pendientes continúan su flujo normal
  • Notificación: Clientes ven mensaje "Compras temporalmente deshabilitadas"

✅ Sistema de Aprobaciones

Flujo administrativo de órdenes

  • Control: Administra qué órdenes requieren aprobación
  • Tiempo real: Cambios aplicados inmediatamente vía WebSocket
  • Riesgo: Útil para control de límites y prevención de fraudes
  • Flexibilidad: Aprobación automática o manual según configuración
